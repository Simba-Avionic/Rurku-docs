\paragraph{Success criteria}
\begin{itemize}
  \item Successful ignition inside the combustion chamber.
  \item Chamber pressure and thrust within expected band for the planned burn
    duration (5\,s nominal).
  \item Shutdown sequence completes without critical hardware damage.
  \item Complete DAQ log and post-test photos.
\end{itemize}

\paragraph{Failure / abort criteria}
\begin{itemize}
  \item Misfire $\rightarrow$ hold; follow misfire wait and approach procedure.
  \item Loss of critical telemetry (chamber pressures, load cell) $\rightarrow$
    HOLD / abort.
  \item Sensor out of limits before ignition $\rightarrow$ HOLD / abort.
  \item Combustion outside the combustion chamber (e.g.\ ejected igniters).
  \item Hard start / overpressure $\rightarrow$ execute D-3 hot abort; safing per
    D-1 / D-2.
\end{itemize}

\subsection*{Report}

\paragraph{Outcome}
Pass (with anomalies) --- analysis dated 21.07.2026.

\paragraph{Results}
Ignition inside the chamber succeeded after the TP-SF-001 igniter remount.
Command timeline in checklist~C was executed as logged; main valves closed at
T$+$7\,s as planned. Visual burn from Dominik's GoPro~4 (\texttt{liquid static 2.mp4},
120\,fps) looked nominally stable for $\approx$1.3\,s after ignition.

\medskip
\noindent\textbf{Timeline vs.\ pressure + video} (ignition $t_0 \approx$ timestamp 504017\,ms
in \texttt{launch2.csv}; analysis window 503279--521043):
\begin{itemize}
  \item \textbf{$t \approx 0$:} Clear ignition on chamber pressure and on video.
  \item \textbf{$0$--$\approx$750\,ms:} Stable operation; chamber $\approx$14.5--16\,bar with a
    short spike near $t \approx 250$\,ms.
  \item \textbf{$\approx$750\,ms:} Chamber pressure begins to fall; ethanol injector /
    manifold pressure rises correspondingly. Not obvious on the film.
  \item \textbf{$\approx$1300\,ms:} Clear loss of thrust / chamber pressure on video and
    on the pressure trace.
  \item \textbf{$\approx$2400\,ms:} Sudden engine ``excitation'' / possible hard transient
    on film; ethanol manifold pressure collapses while chamber pressure stays nearly
    flat.
  \item \textbf{$\approx$2.5--7\,s:} Quasi-equilibrium with low-frequency combustion
    roughness consistent with chugging (chamber $\leftrightarrow$ ethanol hydraulics).
  \item \textbf{$t = 7$\,s:} Planned shutdown; main valves close.
\end{itemize}

\paragraph{Anomalies}
\begin{itemize}
  \item No usable thrust-stand / load-cell recording despite successful calibration
    earlier the same day --- thrust in the results table is estimated, not measured.
  \item Ethanol tank supercharge only $\approx$32.5\,bar vs.\ 40\,bar planned.
  \item Large ethanol feed $\Delta p$ ($\approx$12.37\,bar vs.\ $\approx$4.23\,bar RPA jacket
    expectation) --- suspect channel / injector pressure drop and/or check-valve /
    feed hydraulics.
  \item Chamber pressure decay from $\approx$750\,ms with concurrent ethanol manifold
    rise; visible thrust loss by $\approx$1.3\,s.
  \item Hard transient / possible injector event at $\approx$2.4\,s (manifold pressure
    collapse; film shows sudden plume excitation).
  \item Low-frequency chugging-like instabilities from $\approx$2.5\,s to shutdown.
\end{itemize}

\paragraph{Conclusions / follow-ups}
\begin{itemize}
  \item Investigate nitrogen pressurant path and ethanol supercharge shortfall
    (why 32.5\,bar instead of 40\,bar).
  \item Inspect injector / pintle and check valves after the $\approx$2.4\,s event;
    quantify orifice / channel $\Delta p$.
  \item Fix thrust-stand logging before the next static fire; do not rely on
    estimated thrust for ENG close-out.
  \item Treat chugging and early chamber-pressure decay as open issues before
    packing / flight gate --- consider feed-system damping and O/F / supercharge
    setpoints for the next SF.
\end{itemize}
